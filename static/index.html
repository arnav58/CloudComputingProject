  • Jan'16-May'16

    Privacy Preserving Data Mining


    Wrote python scripts to hide customer's sensitive information in a database and mine the database for interesting data.


    The project is built around the scenario where two or more banks are merging their data to analyse what types of insurances sell more often and with what premium.

    Considering the bank data consists of sensitive customer information which the bank might not want to reveal to the other participating banks, privacy preservation was a necessity.

    This project first pre processes the bank dataset provided, hides the sensitive data using k-Anonimity and Gaussian Noise and then uses Apriori algorithm to generate the frequent itemsets from the data.
    Using the generated itemsets, the managers/data analysts can infer what type of insurance sells more often and what age-group they should target with a particular insurance scheme.
    It also helps them decide what the ideal premium amount should be for each type of insurance.
